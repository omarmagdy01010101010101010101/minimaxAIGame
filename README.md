Tic Tac Toe AI

Project Overview
This project is a simple implementation of the classic Tic Tac Toe game using Python.  
The game allows a human player to compete against an AI opponent.

AI Algorithm
The AI uses the Minimax Algorithm, which evaluates all possible moves and selects the optimal one.  
It ensures that the AI plays in the best possible way to either win or force a draw.

Objectives
- Implement a simple game using Python  
- Apply Artificial Intelligence techniques in games  
- Demonstrate decision-making using Minimax algorithm  

Game Description
- The game is played on a 3x3 grid  
- Player uses X  
- AI uses O  
- The goal is to get three symbols in a row (horizontal, vertical, or diagonal)

Features
- Graphical User Interface using Tkinter  
- Human vs AI gameplay  
- Score tracking system  
- Restart game option  
- Draw detection  

Environment
- Single-player game (Human vs AI)  
- Fully observable environment  
- Deterministic AI (no randomness in decisions)  

How to Run
1. Make sure Python is installed  
2. Run the file:

python game.py

Tech Stack
- Python  
- Tkinter (GUI)  
- Minimax Algorithm (AI)
